STRINGS

Vimos diferentes métodos que se aplican sobre strings, entre otros:

  • string.slice(i,f)//substring(i,f)--> recorta la palabra entre (caracter inicial y final)
  • string.length --> obtiene la cantidad de caracteres del string
  • string.trim() --> elimina los espacios al inicio/final del string
  • string.toUpperCase()//toLowerCase() --> convierte el string en todo mayúsculas/minúsculas
  • string.concat() --> concatena string con lo que queramos
  • string.charAt(x) --> obtiene el caracter en la posición x que consultamos
  • string.split() --> divide el string de acuerdo al separador que le indiquemos

Arrays (arreglos, listas)

Es una estructura de datos que nos permite almacenar varios elementos en una sola variable.

Podemos pensar en esto como una lista ordenada de elementos en la cual cada uno de ellos tiene un índice que indica su posición en la lista.

Podemos almacenar cualquier tipo de datos.

A diferencia de otros lenguajes de programación, el largo del array no está definido de antemano, es dinámico.

Posicionar mouse y retirarlo para transformar este párrafo en un Array

Operaciones y métodos sobre arrays:

  • array.length --> Es la cantidad de elementos del array
  • arrayNumeros.sort() --> Sobrescribe el array original, ordenando los números de menor a mayor.
  • arrayLetras.sort() --> Sobrescribe, ordena alfabéticamente.
  • array[array.length-1] --> Accedo al último elemento del array.
  • Hacer click para traer el array en forma de lista (usando un bucle for):
  • array.push(elem) o array[array.length]= elem --> Agregaría un nuevo elemento al array
  • array.map(nar,manz) --> Transforma mi array de naranjas a manzanas.
  • array.shift() --> Elimina el primer elem. del array y devuelve un nuevo array.
  • array.splice(x,y,'a','b',) --> Inserta en la posición x los elem. 'a','b', etc, eliminando y elementos desde el índice x
  • array.slice(x,y) --> No modifica el array, pero lo corta entre los índices x,y

Aclaración:

Sobre el declarar los arrays como "const" ya que no se pueden redeclarar. Por lo que se recomienda usar let para declararlos.
